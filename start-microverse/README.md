# Start Project
>This project is implementation of tools necessary for productive working template design
>Linter files are used to configure and check for errors in the project
## Built With

- HTML5
- CSS3

### Usage
 - to get a local copy up and running follow these simple example steps.

- Open terminal

- Clone this project by the command `git clone https://github.com/mutinhiri/start-microverse`

- Open `index.html` in your local browser or using Live Server in Visual Studio Code.
